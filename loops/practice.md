- Ćwiczenie 3

Przepisz kod zmieniając pętlę `for` na `while` bez zmiany jego zachowania (wynik powinien pozostać taki sam).

```js
for (let i = 0; i < 3; i++) {
  alert( `number ${i}!` );
}
```
