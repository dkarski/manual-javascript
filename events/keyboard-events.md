| Name  |  Description                                                            |
|----------|---------------------------------------------------------------------------------|
| keypress | ANY key except Shift, Fn, CapsLock is in pressed position. (Fired continously.) |
| keyup    | ANY key is released                                                             |
| keydown  | ANY key is pressed  |
