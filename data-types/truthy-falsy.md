# Truthy/falsy

**Poniższe wartości są zawsze 'fałszywe':**

* false
* 0 (zero)
* '' or "" (posty string)
* null
* undefined
* NaN

Oraz:

* document.all

**Każda pozostała wartość jest zawsze 'prawdziwe', włączając poniższe:**

* '0' (string z wartością zero)
* 'false' (string z wyrazem “false”)
* [] (pusta tablica)
* {} (pusty obiekt)
* function(){} ('pusta' funkcja)
